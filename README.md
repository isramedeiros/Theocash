# 💰 Theocash

> Uma fintech sem complicação. Só o que você precisa e nada mais.

## 📋 Sobre o Projeto

**Theocash** é uma solução fintech que vai contra a corrente do mercado tradicional. Enquanto a maioria dos bancos digitais oferece dezenas de produtos financeiros complexos, o Theocash foca no essencial: **conta digital, transferências, pagamentos e cartão**. 

A proposta é simples: oferecer apenas o necessário para quem quer uma vida financeira descomplicada, sem bombardeio de ofertas de crédito, investimentos mirabolantes ou produtos que você nunca pediu.

### 🎯 Filosofia

- **Minimalismo**: Só o essencial, nada de extras desnecessários
- **Simplicidade**: Interface limpa, processos diretos
- **Transparência**: Sem letras miúdas ou taxas escondidas
- **Controle**: Você no comando das suas finanças, sem empurrões de produtos

## ✨ Funcionalidades

### Core Banking
- ✅ Conta digital gratuita
- ✅ Cartão de débito virtual e físico
- ✅ Transferências TED/DOC
- ✅ PIX (transferências instantâneas)
- ✅ Pagamento de boletos
- ✅ Extrato detalhado

*Foco no básico bem feito.*
